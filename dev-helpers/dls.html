Learning Experiences

Tracking API for learning products. Products are assumed to be composed of learning activities, and users (who are entitled to those products) will generate one or more statements describing interactions/transactions against specific activities. This API provides functions to create, query and manage learning activities and associated statements.

Activity

API for managing activity takes and state. An Activity is a unit of interactive content. An activity has a start-time, end-time and STATE information. It can stay in an incomplete state (paused) for a period of time, and resumed later (after few minutes, hours or days). The STATE information is used to seamlessly resume from where the student left off. When users interact with an ACTIVITY they create “STATE” i.e. subsequent interactions may require knowledge data from previous interactions.

Collaboration

The Collab API provides functions for managing the personal connections for an authenticated user i.e. start following other users and retrieve list of users following you. It also supports operation for posting comments (to a group, or to personal network), retrieving the activity feed the authenticated user.
